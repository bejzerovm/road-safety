# road-safety

Система интеллектуального анализа безопасности дорожного движения на перекрёстках с использованием видеоданных с БПЛА и алгоритмов компьютерного зрения.

## Обзор проекта
Репозиторий `road-safety` содержит программный комплекс для высокоточного мониторинга конфликтных ситуаций. Использование ракурса "вид сверху" (Bird's Eye View) позволяет минимизировать окклюзии и переводить экранные координаты в метрическую систему для объективного расчета параметров движения.

Проект ориентирован не на простой подсчет трафика, а на выявление **косвенных показателей безопасности (Surrogate Safety Measures)**, что позволяет прогнозировать вероятность ДТП до их совершения.

## Ключевые возможности
* **Multi-Model Pipeline**: Гибкая архитектура с поддержкой различных детекторов:
    * **YOLOv8 / YOLOv10**: Оптимальный баланс скорости и точности.
    * **RT-DETR / RF-DETR**: Трансформерные модели для стабильной детекции.
    * **SAHI (Slicing Aided Hyper Inference)**: Технология нарезки кадров для обнаружения микро-объектов на 4K-видео с большой высоты.
* **Анализ безопасности (SSM)**:
    * **TTC (Time-to-Collision)**: Расчет времени до потенциального столкновения.
    * **PET (Post-Encroachment Time)**: Оценка временного интервала между прохождением объектов через одну зону.
    * **PSD (Proportion of Stopping Distance)**: Анализ достаточности дистанции для экстренного торможения.
    * **Shockwave Analysis**: Обнаружение ударных волн торможения в потоке.
* **Геометрия и Проекция**: Преобразование координат через матрицы гомографии для получения реальных скоростей (км/ч) и расстояний (м).

## Стек технологий
* **Core**: Python 3.10+
* **CV Engines**: OpenCV, Supervision, Ultralytics
* **Tracking**: ByteTrack, BoT-SORT
* **Analysis**: NumPy, Pandas, SciPy
* **Visualization**: Matplotlib, Seaborn, Plotly

## Структура репозитория
* `/configs` — файлы конфигурации моделей и параметров анализа.
* `/data` — примеры видеоматериалов с дронов и калибровочные сетки.
* `/src/detection` — модули детекции с использованием различных архитектур.
* `/src/tracking` — алгоритмы фильтрации и ведения траекторий.
* `/src/analytics` — ядро системы: расчет метрик TTC, PET и уровня опасности.
* `/viz` — скрипты генерации Heatmaps, Space-Time Cubes и отчетов.
